# F.A.Q.

1. Q: I didn't find my board on the list.

    A_1: Open **"boards.db"** with text editor and add it manually ***(instructions included)***.

    A_2: Open new issue **with template**.

2. Q: I found a bug.

   A: Open new issue **with details**.
